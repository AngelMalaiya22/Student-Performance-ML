**Student Performance Prediction**

📌 Project Overview This project predicts student final grades (G3) using machine learning models.


📂 Dataset The dataset contains student-related information such as:

Study time
Family background
Previous grades (G1, G2)
Absences
Lifestyle factors
⚙️ Workflow

Data Cleaning
Feature Encoding (get_dummies)
Feature Scaling (StandardScaler)
Train-Test Split
Model Training
Model Evaluation
🤖 Models Used

Linear Regression
Decision Tree
Random Forest
📊 Evaluation Metrics

Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R² Score
🏆 Results Linear Regression performed best with:

R² Score ≈ 0.79
📈 Visualization Bar graph used to compare model performance.
